📱 How to export from iPhone / iCloud:
  1. On your iPhone: Settings → [Your Name] → iCloud → make sure Contacts is toggled ON (this syncs all phone contacts to iCloud)
  2. On a computer, go to icloud.com and sign in
  3. Click Contacts
  4. Press Cmd+A (Mac) or Ctrl+A (PC) to select all contacts
  5. Click the gear icon ⚙️ at the bottom left → Export vCard
  6. A .vcf file downloads — tap the button below to upload it

📧 About email sync
Live email sync (Gmail, Outlook, iCloud reading emails automatically) requires a backend server — browsers can't connect directly to email APIs for security reasons. That's a proper development project.

What works right now:
• Import contacts from any account using the file upload above
• Log notes and communications manually inside each contact record
• Use the BCC method: BCC crm@havens.app on emails you want logged
⭐ All imported contacts are automatically tagged yellow (New) for 30 days.
